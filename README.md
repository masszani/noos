# NoAsAService
No As A Service

Usage:

NOOIZE Method

POST http://localhost:8080/nooize

POST SAMPLE

{
	"Testo": "certo",
	"Tipo": "G"
}

Where Tipo:
N -> NO
G -> Giusto
E -> Eh

POST http://localhost:8008/autog

POST SAMPLE

{
	"Testo": "certo"
}
